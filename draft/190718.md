# Apache FileMatchs 学习


```
# 凡是匹配到 zip,gz,rar,box,log结尾的文件，进行下面的规则进行匹配
<filesmatch ".(zip|gz|rar|box|log)"> 
Order Allow,Deny  //匹配顺序 Allow,Deny
Deny from all  //拒绝所有
# Allow from env=local_ref //允许标记为local_ref的Referer
</filesmatch>
```
